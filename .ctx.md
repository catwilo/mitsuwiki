# mitsuwiki — context
## status: ACTIVE — migración en curso

Personal wiki. Markdown notes, offline browser SPA, terminal search via fzf.
Source of truth: ~/unix-toolkit-tools/mitsuwiki/

## structure
- wiki.sh        — CLI entrypoint (serve|stop|search|open|new|index)
- install.sh     — symlink wiki.sh -> ~/.local/bin/wiki (idempotente)
- index.html     — SPA dark theme, sidebar + marked.js renderer
- marked.min.js  — bundled offline, sin CDN (~40KB)
- notes.json     — auto-generado por wiki serve/index (NO commiteado)
- notes/         — todas las notas .md por categoría
- assets/        — PDFs de referencia

## commands
  wiki serve [port]    byobu window "mitsuwiki" + python3 http.server + clipso URL
  wiki stop            kill byobu window "mitsuwiki"
  wiki search [query]  fzf sobre contenido .md -> $EDITOR
  wiki open            fzf picker -> $EDITOR
  wiki new <cat/name>  crea .md desde template -> $EDITOR
  wiki index           rebuild notes.json manualmente

## integrations
- byobu: server en ventana "mitsuwiki"
- clipso: URL copiada al clipboard en serve
- fzf: search + open picker
- $EDITOR (nvim): todo el editing
- alias wiki en dotconfigtermux/shared.zsh

## instalación (Termux)
- ~/.local/bin/wiki -> ~/unix-toolkit-tools/mitsuwiki/wiki.sh OK
